## 2. 한국관광공사_무장애 여행 정보 API

- **참고 URL**: https://www.data.go.kr/data/15101897/openapi.do
- **역할**: 무장애 관련 핵심 데이터(접근성, 휠체어 등) 및 관광지 탐색 기능을 위한 주력 API

| 활용 기능명                         | API 오퍼레이션명     | 활용 목적                                                    |
| ----------------------------------- | -------------------- | ------------------------------------------------------------ |
| **무장애 정보**                     | `detailWithTour2`    | 관광지·숙박·음식점 등의 장애인 화장실, 주차장, 경사로, 엘리베이터 등 **기본 접근성 정보를 구축하는 핵심 데이터**로 활용한다. |
| **위치기반관광정보**                | `locationBasedList2` | 사용자 현재 위치 주변의 관광지와 무장애 시설 후보를 조회하는 데 활용한다. 주변 장소 탐색과 장소 진입 확인의 기준 데이터로 사용한다. |
| **지역기반관광정보**                | `areaBasedList2`     | 지역별 무장애 관광지 목록을 구성하는 데 활용한다. 시범 지역 선정, 지역 특화 관광지 탐색, 지역별 접근성 정보 구축의 기반 데이터로 사용한다. |
| **키워드검색**                      | `searchKeyword2`     | 사용자가 관광지명, 시설명, 지역명을 직접 검색할 때 활용한다. 검색 결과를 장소 상세 화면과 무장애 정보 조회로 연결한다. |
| **공통정보**                        | `detailCommon2`      | 장소명, 주소, 좌표, 개요 등 시설 상세 페이지의 기본 정보를 구성하는 데 활용한다. 사용자 제보가 연결될 기준 장소 데이터로 사용한다. |
| **소개정보**                        | `detailIntro2`       | 이용시간, 휴무일, 주차시설 등 방문 전 확인이 필요한 정보를 제공하는 데 활용한다. 접근성 정보와 함께 실제 방문 가능 여부를 판단하는 보조 정보로 사용한다. |
| **반복정보**                        | `detailInfo2`        | 관광지·숙박·여행코스별 세부 정보를 보완하는 데 활용한다. 향후 교통약자 맞춤 여행코스 추천 기능과 연계할 수 있다. |
| **이미지정보**                      | `detailImage2`       | 시설 상세 화면의 기본 사진을 제공하는 데 활용한다. 향후 사용자 크라우드소싱 사진과 함께 제공하여 공식 이미지와 현장 이미지를 병행 표시한다. |
| **무장애 여행정보 동기화 목록정보** | `areaBasedSyncList2` | 공공데이터의 변경 여부를 확인하고 내부 데이터를 갱신하는 데 활용한다. 정기 동기화를 통해 기본 관광지·무장애 정보를 최신 상태로 유지한다. |

## 3. 한국관광공사_국문 관광정보 서비스_GW

- 참고 URL: https://www.data.go.kr/data/15101578/openapi.do
- 역할: 무장애 여행 정보 API만으로는 부족한 일반 관광지로서의 부가 데이터(사진, 소개 등)를 보완하는 API

| 활용 기능명    | API 오퍼레이션명 | 활용 목적                                                    |
| -------------- | ---------------- | ------------------------------------------------------------ |
| **공통정보**   | `detailCommon2`  | 무장애 여행 정보만으로 부족한 장소명, 주소, 좌표, 개요 등 **일반 관광지 기본 정보를 보완**하는 데 활용한다. |
| **소개정보**   | `detailIntro2`   | 관광지 유형별 이용시간, 휴무일, 주차시설 등 방문 전 확인이 필요한 일반 관광 정보를 보완하는 데 활용한다. |
| **이미지정보** | `detailImage2`   | 관광지 상세 화면의 기본 사진을 보강하는 데 활용한다. 무장애 정보 API에서 부족한 이미지 자료를 보완하고 사용자 현장 사진과 병행 제공한다. |
